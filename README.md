# SALT Dance Academy

Website for **SALT Dance Academy**, Mansarovar, Jaipur — React, Vite, Framer Motion, and SCSS modules.

**Tagline:** Sweat. Dedication. Attitude.  
**Location:** Shop No. 4 & 9, Above Adidas Showroom, Main Madhyam Marg, VT Road Circle, Mansarovar, Jaipur 302020  
**Phone:** +91 98877 39989 · +91 78918 70134

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 19 + Vite
- React Router
- Framer Motion
- SCSS Modules
